# HR Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes employee data to provide insights into workforce demographics, attrition, job roles, and employee performance.

## Objectives
- Analyze employee attrition
- Track workforce demographics
- Evaluate job role distribution
- Monitor employee satisfaction
- Support HR decision-making

## Tools Used
- Power BI
- Excel
- GitHub
